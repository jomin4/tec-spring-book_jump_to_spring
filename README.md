# 점프 투 자바


단순히 책의 예제를 따라 치는 것에 그치지 않고, 책을 읽으며 생긴 궁금증을 직접 코드로 검증하고 실험하는 과정을 함께 기록합니다.

---

## 학습 방식

이 저장소는 크게 두 가지 방식으로 학습을 진행합니다.

### 1. 책 기반 학습

기본 학습은 **점프 투 스프링 부트(Jump To Spring Boot)** 내용을 따라가며 진행합니다.

* 책에서 제공하는 예제 구현
* 핵심 개념 정리
* 단계별 실습 코드 작성
* 웹 애플리케이션 동작 원리 학습

---

### 2. 추가 탐구 학습

책을 읽다가 궁금한 내용이 생기면 별도의 실습 코드를 작성하여 직접 검증합니다.

예시

* Cookie 동작 원리
* Session 관리 방식
* 로그인 / 로그아웃 처리
* 권한 인증(Authentication / Authorization)
* Servlet 동작 과정
* HTTP 요청 / 응답 흐름

단순히 개념을 암기하는 것이 아니라 실제 요청을 발생시키고 데이터를 추적하며 내부 동작을 이해하는 것을 목표로 합니다.

---

## 학습 목표

이 프로젝트의 목표는 단순히 Spring Boot를 사용하는 것이 아닙니다.

다음과 같은 질문에 스스로 답할 수 있는 수준을 목표로 합니다.

* 브라우저 요청은 어떻게 서버까지 도착하는가?
* Servlet Container는 어떤 역할을 하는가?
* Cookie는 어디에 저장되고 언제 전송되는가?
* Session은 어떻게 로그인 상태를 유지하는가?
* Spring MVC는 Servlet 위에서 어떻게 동작하는가?
* Spring Security는 내부적으로 무엇을 처리하는가?

---

## 프로젝트 구조

```text
src/main/java/com/back

├── book
│   └── 점프 투 스프링 부트 실습 코드

└── extra
    ├── cookie
    ├── session
    ├── login
    └── auth
```

* `book` : 책 내용을 따라가는 실습 코드
* `extra` : 추가 실험 및 내부 동작 검증 코드
